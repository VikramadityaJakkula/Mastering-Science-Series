THEORY OF MULTIPLICATION 
 Definition:  Multiplication is a short way to find the sum of a number a certain amount of times.

  For example, 2 x 5 = 10 instead of 2 + 2 + 2 + 2 + 2 = 10.

    * We multiply a number called ‘Multiplicand’ by a number called ‘Multiplier’ and get the result 
    called ‘Product’.

                    15         x       5              =             75

            Multiplicand        Multiplier                   Product

    * The multiplicand and multiplier are also called factors of the product. We multiply two factors
     and get the ‘Multiple’. In the multiplication shown below the number 75 is the multiple of both
      5 and 15. At the same time 15 and 5 are factors of 75.

                    15         x       5              =             75

                Factor                Factor                    Multiple

    * In multiplication the order of numbers doesn’t matter. We may multiply numbers in any order the
     product will remain the same.
STEPS INVOLVED

We will discuss the method of finding the product of two numbers.

Lets take up an example :    

                       206
                     X 314 
MULTIPLICATION BY EXPANSION
 Expand : 206 = 2 Hundreds + 0 Tens + 6 Ones = 200 + 0 + 6

Step 1: Multiply the Ones 
The Ones place value in 206 is 6.
314 X 6 X 1 = 1884

Step 2: Multiply the Tens 
The Tens place value in 206 is 0.
314 X 0 X 10 = 0

Step 3: Multiply the Hundreds 
The Hundreds place value in 206 is 2.
314 X 2 X 100 = 62800

Answer : 62800 + 0 + 1884 = 64684
                                
Now , we will try to simplify the process of performing the multiplication by stacking the numbers in their
 place values.

MULTIPLY BY STACK
To stack numbers::
    - Place the numbers you want to add on top of each other in their value places.
    - Place the plus sign, X , on the left of the stack
    - Draw a line at the bottom.
    
                                	                     206
                                     	   	  X 314
                                             		      ?
                            206
                         X 314
                          1884 <-  (314 X 6 X 1) 
                          0000 <-  (314 X 0 X 10)
                         62800 <-  (314 X 2 X 100)
                         64684
 Explanation:

206 x 314 =?

Ones Place (314 x 6):
                    (1) 4 x 6 = 24
                         Take "4" and carry "2"
                    (2) 1 x 6 = 6 (add the carry : 2) = 6 + 2 = 8 
                    (3) 3 x 6 = 18
Answer: 314 x 6 = 1884

Tens Place (314 x 0):
                    (1) 4 x 0 = 0
                    (2) 1 x 0 = 0   
                    (3) 3 x 0 = 0
Answer: 314 x 0 = 0

Hundreds Place (314 x 2):
                    (1) 4 x 2 = 8
                    (2) 1 x 2 = 2
                    (3) 3 x 2 = 6
Answer: 314  x 2 = 628

Add : (1884 X 1 + 0 X 10 + 628 X 100):
       1884 + 0 + 62800 = 64684.
