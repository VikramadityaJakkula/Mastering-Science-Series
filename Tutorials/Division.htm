INTRODUCTION 
 Example, 115 ÷ 2

     A number which is to be divided is called dividend, the number by which we divide is called 'divisor', 
    the result of division is called quotient and that which is left after division is called remainder.
STEPS INVOLVED

Evaluate: 

     885 
  ÷ 9
So, 885 ÷ 9 = 98, remainder 3 
Detailed Steps :
Use the Long Division to solve . Each step has Division,Multiplication,followed by Subtraction.[DMS]

Expand: 885 = 8 Hundreds + 8 Tens + 5 Ones = 800 + 80 + 5

Step 1: Consider the Hundreds place value.
Old Remainder = 0
New Dividend = Old Remainder X 10 + Hundreds Place Value = 0 X 10 + 8 = 8
D: Number of Hundreds in Quotient = New Dividend / Divisor = 8 / 9 = 0
M: 9 X 0 = 0 
S: Remainder = 8 - 0 = 8 

Step 2: Consider the Tens place value.
Old Remainder = 8
New Dividend = Old Remainder X 10 + Tens Place Value = 8 X 10 + 8 = 88
D: Number of Tens in Quotient = New Dividend / Divisor = 88 / 9 = 9
M: 9 X 9 = 81 
S: Remainder = 88 - 81 = 7 

Step 3: Consider the Ones place value.
Old Remainder = 7
New Dividend = Old Remainder X 10 + Ones Place Value = 7 X 10 + 5 = 75
D: Number of Ones in Quotient = New Dividend / Divisor = 75 / 9 = 8
M: 9 X 8 = 72 
S: Remainder = 75 - 72 = 3 

Quotient = 0 Hundreds + 9 Tens + 8 Ones = 0 + 90 + 8 = 98
Remainder = 3
